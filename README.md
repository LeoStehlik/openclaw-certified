# OpenClaw Certified Architect - Foundations

> There is no actual certification. Yet.

Study materials for the **OpenClaw Certified Architect - Foundations** track.

Build production-grade AI harnesses with OpenClaw. Six modules covering workspace engineering, agent orchestration, memory architecture, skill development, production ops, and sprint engineering.

No drag-and-drop workflows. No certificate at the end. Just the knowledge.

---

| # | Module | What You Learn |

| 01 | [Harness Engineering](modules/01-harness-engineering.md) | Workspace structure, AGENTS.md, SOUL.md, identity, bootstrap |
| 02 | [Agent Orchestration](modules/02-agent-orchestration.md) | sessions_spawn, subagents, session routing, ACP harnesses |
| 03 | [Memory Architecture](modules/03-memory-architecture.md) | MEMORY.md, daily logs, AutoDream, open-items, memory hygiene |

| 06 | [Sprint Engineering](modules/06-sprint-engineering.md) | proof-loop, brief-master, blast radius, AC-driven development |

## Who This Is For

- Developers building personal AI harnesses on OpenClaw
- Teams deploying multi-agent workflows for real products
- Anyone tired of toy demos and drag-and-drop AI builders

---

## Prerequisites

- OpenClaw installed and running
- At least one working Telegram or Discord channel connected
- Basic familiarity with Claude / LLMs

---

## How to Use

Work through the modules in order. Each module has:
- **Concept** - what it is and why it matters
- **How it works** - mechanics and config
- **Exercises** - hands-on tasks to actually build the thing
- **Common mistakes** - what trips people up

No passive reading. Build as you go.
